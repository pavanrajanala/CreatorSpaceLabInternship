# Creator Space Lab Internship

---

## 📌 Overview

This repository contains the work completed as part of the Creator Space Lab Internship program.  
The project focuses on **Python programming, SQL queries, Exploratory Data Analysis (EDA), and Machine Learning implementation** using real-world datasets.

The objective is to strengthen practical skills in data analysis, statistical thinking, and machine learning workflows.

---

## 🧮 1. Python Programming Tasks

### 📌 Student Marks Analysis
A Python program designed to analyze student performance.

**Features:**
- Store marks of 5 students using a list
- Compute:
  - Highest mark
  - Lowest mark
  - Average mark
  - Count of students above average

---

### 📌 Employee Salary Management
A dictionary-based program for employee salary analysis.

**Features:**
- Store employee names and salaries
- Compute:
  - Highest salary employee
  - Lowest salary employee
  - Average salary
  - Employees earning above average salary

---

## 🗄️ 2. SQL Query Tasks

### 📌 Employee Table Analysis

### 📌 Sales Table Analysis
